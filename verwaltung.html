Benutzerkonten anlegen

Neue Konten gibt es nur über das Supabase-Dashboard – dafür brauchst du deinen eigenen Supabase-Projektzugang (getrennt vom Login hier in der App).

Supabase-Dashboard → Authentication → Users öffnen ↗

Benutzername-Konto für eine andere Person

  1. Im Dashboard auf Add user klicken.
  2. Als E-Mail eintragen: <gewünschter-benutzername>@benutzer.mdr-datenbank.local (die Domain muss genau so geschrieben werden).
  3. Ein Passwort vergeben und speichern.
  4. Der Person nur den Benutzernamen (ohne die @…-Domain) und das Passwort mitteilen – damit meldet sie sich auf der Login-Seite an.

Konto löschen / Passwort ändern

Im Dashboard bei der jeweiligen Person auf die drei Punkte klicken → Delete user bzw. Reset password.

Dein eigener Admin-Zugang

Dein Konto ist das einzige mit einer echten E-Mail-Adresse statt der internen Benutzername-Domain – daran erkennt die App, dass du Admin bist und diese Seite sehen darfst.
